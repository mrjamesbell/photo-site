Quick Start

To view the gallery with placeholder images:

  1. Open public/index.html directly in your browser, or
  2. Use a local server (recommended for module imports to work correctly)

Option 1: Simple File Access

Click the button below to open the gallery:

Open Gallery (File)

Option 2: Local Server (Recommended)

For the best development experience, use a local server:

Using Python:

cd public
python3 -m http.server 8000

Then visit: http://localhost:8000

Using Node.js (http-server):

npx http-server public -p 8000

Then visit: http://localhost:8000

Using PHP:

cd public
php -S localhost:8000

Then visit: http://localhost:8000

üìù Note

The gallery will automatically use mock data with placeholder images when running locally. This allows you to see the design and functionality without needing to set up Cloudflare infrastructure.

When deployed to Cloudflare Pages, it will automatically use the real API endpoints.
